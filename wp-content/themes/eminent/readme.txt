=== About Eminent ===
Eminent is a modern looking premium quality multipurpose WordPress theme for every type of businesses for their online presence. Eminent is mobile responsive, SEO-friendly and optimized for quick loading in web browsers. Eminent has several options to customize the home page text, logo etc. footer

== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "Eminent.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (Eminent.zip) that you have downloaded.
 = Upload the entire folder (Eminent) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/Eminent/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "Eminent" to activate the theme.

5. one custom widgets :
	i) Eminent social widget for display socal icons in footer.
	ii) Eminent Get in touch widget for display the contact information in footer

6. Theme Customizer Options:	

	a) Basic Settings:
           You can manage here site logo, favicon, breadcrumb background image, blog title and copyright text.	

	b) Home Page Settings:
		i)   Slider Section	: You can add Soliloque Slider Id.
		ii)  Welcome Section : You can set Welcome section title, subtitle, description and background image.
        	iii) Blog Section : you can select category of posts and add title, description of posts slider.
                
    	c) Social Settings:
            You can add here social media links.
	
	d) Contact Settings:
            You can add here contact number and email.
